# Results

This folder contains selected final result tables and figures used in the project report.

Large intermediate files are not included in this repository.

## Figures

The `figures/` folder contains selected visualization outputs from:

- preprocessing quality control;
- taxonomic profiling;
- functional and pathway-level analyses;
- metabolomics association analysis;
- cross-omics analysis;
- Random Forest classification.

## Tables

The `tables/` folder contains selected summary tables from:

- HUMAnN quality control;
- SQ pathway checks;
- MaAsLin2 association analyses;
- sulfur-related cross-omics analyses;
- Random Forest model comparison.

Full raw data, complete HUMAnN/MetaPhlAn outputs, and large intermediate matrices are not stored in this repository.
